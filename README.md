# Denvan Hair Braiding

Professional hair braiding website for Denvan Hair Braiding, serving the Dallas-Fort Worth metroplex with salon and mobile services.

## About

Denvan Hair Braiding offers expert braiding services from a Frisco, TX salon and mobile service across DFW. All services are by appointment only. Client satisfaction is our goal.

## Features

- **Mobile-first responsive design** — optimized for phones, tablets, and desktops
- **Gallery with starting prices** — curated photos organized by category with visible starting prices
- **Send This Style** — WhatsApp integration lets clients send a style photo to book instantly
- **Image protection** — watermark overlay, right-click disabled, drag disabled, print blocked
- **Newsletter subscribe** — email signup via FormSubmit.co on homepage and gallery
- **Lightbox viewer** — full-screen image viewing with watermark and WhatsApp send button
- **Trust signals** — location, experience, client count, and testimonials
- **WhatsApp chat button** — floating button on every page for instant messaging
- **Lazy loading** — gallery images load on scroll for fast page performance
- **Scroll animations** — fade-in effects on scroll using IntersectionObserver
